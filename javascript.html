JavaScript: Powering Interactivity

Discover how JavaScript brings dynamic behaviour and interactivity to web applications.

Theoretical Concepts of JavaScript

What is JavaScript?

JavaScript is a lightweight, interpreted programming language that runs directly in the browser. It is the only programming language that browsers understand natively, which makes it an essential skill for any web developer. JavaScript allows you to make web pages dynamic and interactive — responding to user actions, updating content without reloading, validating form data, and much more. It was originally created for client-side scripting but with the introduction of Node.js it can also run on the server side, making it a full stack language.

DOM Manipulation

The Document Object Model (DOM) is the browser's representation of the HTML page as a tree of objects. JavaScript can access and modify this tree to dynamically change the content, structure, or styling of the page without reloading it. Common methods for selecting elements include document.getElementById(), document.querySelector(), and document.querySelectorAll(). Once you have a reference to an element, you can change its text (innerText), HTML content (innerHTML), or CSS styles (style).

Event Handling

Events are actions or occurrences that happen in the browser — things like a user clicking a button, moving the mouse, pressing a key, or submitting a form. JavaScript listens for these events using addEventListener(), which takes the event type (e.g., 'click', 'submit', 'keypress') and a function to run when the event happens. This event-driven model is fundamental to how interactive web pages work.

Variables, Functions and Conditionals

JavaScript supports all the core programming concepts. Variables are declared with let (for values that will change) or const (for values that will not change). Functions are reusable blocks of code defined with the function keyword or as arrow functions (() => {}). Conditionals (if, else if, else) allow the code to make decisions based on conditions.

Asynchronous Programming

JavaScript is single-threaded, but it handles tasks that take time (like fetching data from a server) asynchronously so the rest of the page does not freeze. This is done using callbacks, Promises, or the modern async/await syntax. The fetch() API is the standard way to make HTTP requests from JavaScript to retrieve or send data to a server without reloading the page.

JavaScript and Cyber Security

From a Cyber Security perspective, JavaScript is important to understand because it is at the centre of many web vulnerabilities. Cross-Site Scripting (XSS) attacks involve injecting malicious JavaScript into a webpage to steal data or hijack sessions. Understanding how JavaScript runs in the browser, how it accesses the DOM, and how it communicates with servers helps in both identifying and preventing these kinds of attacks.

Event Listener – Toggle Colour

Using addEventListener to toggle a box's background colour between two values when clicked.

document.getElementById('toggleColor').addEventListener('click', () => {
  const box = document.getElementById('colorBox');

  if (box.style.backgroundColor === 'rgb(42, 80, 176)') {
    box.style.backgroundColor = '#1e3a8a';
  } else {
    box.style.backgroundColor = '#2a50b0';
  }
});

Form Validation

Using JavaScript to validate form input before it is submitted, preventing empty data from being sent.

document.getElementById('sampleForm').addEventListener('submit', (e) => {
  e.preventDefault();

  const username = document.getElementById('username').value;
  const message  = document.getElementById('formMessage');

  if (username.trim() === '') {
    message.style.color = 'red';
    message.innerText = 'Please enter a username before submitting.';
  } else {
    message.style.color = 'green';
    message.innerText = 'Username "' + username + '" submitted successfully!';
  }
});

Dynamic List Builder

Creating new HTML elements dynamically and adding them to the page using createElement() and appendChild().

document.getElementById('addItem').addEventListener('click', () => {
  const input = document.getElementById('listInput');
  const list  = document.getElementById('dynamicList');

  if (input.value.trim() !== '') {
    const li = document.createElement('li');
    li.innerText = input.value;
    list.appendChild(li);
    input.value = '';
  }
});
