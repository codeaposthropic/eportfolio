CSS: Styling the Web

Explore how CSS transforms web pages with style, layout, and responsive design.

Theoretical Concepts of CSS

What is CSS?

CSS stands for Cascading Style Sheets. It is a stylesheet language used to control the visual presentation of HTML elements on a webpage. While HTML provides the structure and content, CSS handles everything related to how that content looks — including colours, fonts, spacing, borders, backgrounds, and layout. Without CSS, every webpage would look like a plain text document with default browser styling. CSS allows developers to separate the design from the content, which makes websites much easier to maintain and update.

Selectors and Specificity

CSS selectors are patterns used to target specific HTML elements and apply styles to them. The most common selectors are element selectors (e.g., p targets all paragraph tags), class selectors (e.g., .card targets elements with that class), and ID selectors (e.g., #header targets the element with that ID). Specificity is a set of rules that determines which CSS rule wins when multiple rules target the same element. ID selectors have higher specificity than class selectors, and class selectors have higher specificity than element selectors.

Box Model

The CSS box model is a fundamental concept that describes how every HTML element is structured as a rectangular box. This box has four layers from the inside out: content (the actual text or image), padding (space between the content and the border), border (a line around the padding), and margin (space outside the border that separates the element from other elements). Understanding the box model is essential for controlling layout and spacing on a webpage.

Flexbox and Grid

Flexbox (Flexible Box Layout) and CSS Grid are two modern layout systems that make it much easier to design complex, responsive layouts. Flexbox works in one dimension — either as a row or a column — and is great for aligning items along a single axis. CSS Grid works in two dimensions (rows and columns) and is better suited for larger page layouts. Both systems are widely supported in modern browsers and have largely replaced older layout techniques like using floats or tables for layout purposes.

Responsive Design with Media Queries

Media queries allow CSS styles to be applied only under certain conditions, most commonly based on screen size. For example, you can write CSS that changes the layout from three columns on a desktop to a single column on a mobile phone. This technique is central to responsive web design — building websites that look good on all screen sizes without creating separate mobile and desktop versions.

CSS Transitions and Animations

CSS transitions allow property changes to happen smoothly over time rather than instantly. For example, a button can gradually change colour when hovered over. CSS animations allow more complex, multi-step movements to be defined using @keyframes. These features add polish to a design without needing any JavaScript.

Practical Examples of CSS

Styling Text

Customising text with font family, colour, and size using CSS properties.

p {
  font-family: 'Georgia', serif;
  color: #1e3a8a;
  font-size: 1.2rem;
}

Flexbox Layout

A responsive row of items using Flexbox for alignment and spacing.

.flex-container {
  display: flex;
  justify-content: space-around;
  gap: 10px;
  flex-wrap: wrap;
}

.flex-item {
  background-color: #1e3a8a;
  color: #ffffff;
  padding: 12px 20px;
  border-radius: 4px;
  flex: 1;
  min-width: 80px;
  text-align: center;
}

Hover Effects

Adding a smooth hover effect to a button using transition and transform.

.hover-btn {
  background-color: #1e3a8a;
  color: #ffffff;
  padding: 10px 24px;
  border: none;
  border-radius: 4px;
  cursor: pointer;
  transition: transform 0.3s ease, background-color 0.3s ease;
}

.hover-btn:hover {
  transform: scale(1.08);
  background-color: #2a50b0;
}

CSS Grid Layout

Creating a two-dimensional grid layout using CSS Grid with three equal columns.

.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 10px;
}

.grid-item {
  background-color: #1e3a8a;
  color: #ffffff;
  padding: 12px;
  border-radius: 4px;
  text-align: center;
}
