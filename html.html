HTML: The Backbone of the Web

Discover the power of HTML in structuring content and building the foundation of modern websites.

Theoretical Concepts of HTML

What is HTML?

HTML stands for HyperText Markup Language. It is the standard language used to create and structure content on the web. HTML uses a system of elements represented by tags to define different types of content such as headings, paragraphs, links, images, and forms. Every webpage you visit is built on an HTML foundation — it is essentially the skeleton of any website, providing the raw structure that browsers then display to users.

Semantic HTML

Semantic HTML refers to using HTML elements that clearly describe their meaning and purpose, both to the browser and to the developer reading the code. Instead of using generic container elements like <div> for everything, semantic HTML encourages the use of meaningful tags such as <header>, <footer>, <article>, <section>, and <nav>. This improves accessibility for screen readers, makes the code easier to read and maintain, and helps search engines better understand the structure and content of a page.

Attributes and Elements

HTML elements are defined by tags — an opening tag like <p> and a closing tag like </p> wrap the content. Attributes are additional pieces of information added inside the opening tag to configure or describe the element further. For example, the href attribute on an anchor tag specifies where the link goes, and the alt attribute on an image tag provides a text alternative for screen readers. Common attributes include id, class, src, href, and style.

Forms and Input

HTML forms are used to collect information from users. A form is created with the <form> element and can contain various input types such as text fields (<input type="text">), email fields (<input type="email">), dropdowns (<select>), text areas (<textarea>), and submit buttons (<button>). Forms are essential for interactions like logging in, signing up, searching, and submitting data. From a Cyber Security perspective, forms are also one of the most common attack surfaces — things like SQL injection and XSS often target improperly handled form inputs.

Media Embedding

HTML supports embedding various types of media directly into a webpage. The <img> tag is used for images, the <video> tag for video content, and the <audio> tag for sound. External content like YouTube videos or maps can be embedded using the <iframe> element. Always including an alt attribute on images is important for accessibility.

Tables

HTML tables are used to display structured, tabular data. A table is built with <table>, rows with <tr>, header cells with <th>, and data cells with <td>. For accessibility, tables should include a <caption> and use the scope attribute on header cells so screen readers can properly interpret the data.

Practical Examples of HTML

Semantic Structure

A well-structured HTML page using semantic elements for better organisation and accessibility.

<header>
  <h1>Welcome to Aakash's Portfolio</h1>
</header>

<nav>
  <ul>
    <li><a href="index.html">Home</a></li>
    <li><a href="html.html">HTML</a></li>
    <li><a href="css.html">CSS</a></li>
  </ul>
</nav>

<main>
  <article>
    <h2>About Me</h2>
    <p>I am a Cyber Security student learning full stack development.</p>
  </article>

  <section>
    <h2>My Skills</h2>
    <ul>
      <li>Network Security</li>
      <li>Ethical Hacking</li>
      <li>HTML and CSS</li>
    </ul>
  </section>
</main>

<footer>
  <p>© 2026 Aakash Bhandari</p>
</footer>

Interactive Form

A simple HTML form demonstrating how user input is collected on a webpage.

<form action="#" method="post">

  <label for="name">Full Name:</label>
  <input type="text" id="name" name="name" placeholder="Enter your name" required />

  <label for="email">Email Address:</label>
  <input type="email" id="email" name="email" placeholder="Enter your email" required />

  <label for="message">Message:</label>
  <textarea id="message" name="message" rows="4" placeholder="Write your message..."></textarea>

  <button type="submit">Submit</button>

</form>

Accessible Table

A data table built with proper accessibility features including a caption and scoped header cells.

<table>
  <caption>Student Skills Table</caption>
  <thead>
    <tr>
      <th scope="col">ID</th>
      <th scope="col">Skill</th>
      <th scope="col">Level</th>
    </tr>
  </thead>
  <tbody>
    <tr><td>1</td><td>Network Security</td><td>Intermediate</td></tr>
    <tr><td>2</td><td>Ethical Hacking</td><td>Beginner</td></tr>
    <tr><td>3</td><td>HTML and CSS</td><td>Intermediate</td></tr>
    <tr><td>4</td><td>JavaScript</td><td>Beginner</td></tr>
  </tbody>
</table>
